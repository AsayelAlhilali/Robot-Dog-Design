# Robot Dog Mechanical Design

## Overview
This project presents a simple mechanical design of a robot dog, The design aims to provide a stable and efficient foundation for future robot development.

# Robot Body and Chassis 
The robot has a rectangular PLA chassis with rounded corners that supports most of the main components, such as the battery and motors. Each of the four legs is attached to one corner of the chassis, for walking and balance.
