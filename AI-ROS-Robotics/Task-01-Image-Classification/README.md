# Task 01 - Image Classification

## Overview
This project is about image classification model created using **Teachable Machine** and **Python**. The model classifies input images into one of two animal Lion 🦁 or Cheetah 🐆

## Files
- `Image Classification with Teachable Machine.ipynb` – Python code.
- `keras_model.h5` – image classification model.
- `labels.txt` – Class labels.
- `image_test.jpeg` – test image.
- `image_test_result_output.jpeg` – Screenshot of the prediction output.

## Requirements
- Python 3
- TensorFlow / Keras
- NumPy
- Pillow

## Output
The program predicts successfully the class of the test image **Class: cheetah** and displays the **Confidence Score: 0.9996562**.
